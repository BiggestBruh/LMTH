What is CSS?

CSS stands for Cascading Style Sheet. It's a style sheet language (you could say it's a programming language if you wanted to offend a programmer) which allows you to make your web pages look like they have been made later than the 1990s.

Cool Things About CSS


The visibility property can be set to visible (to make an object visible, unsurprisingly) or hidden (to make it invisible but still have it occupy space. Turns out it can also be set to collapse, which works the same as hidden for everything except table rows, columns and groups of rows/columns, which will NOT occupy space when affected by the collapse property.

Speaking of tables, border property allows you to set border-style, border-width and border-color for a table. All 3 of them are actually properties of their own, which means that the border property manipulates other properties, rather than the actual table you're working with. Turns out border-style, border-width and border-color properties can be set separately as well and can be different for each border. A cool trick to know but you probably won't ever use it if you have any respect for yourself or the people viewing your site.

Horizontal percentages are calculated based on container's width, which means an image the width of which is 75% of a 100x200 container is 75 pixels wide. Pretty simple. But for some reason, vertical percentages also use the width of the container. An image the height of which is 75% of a 100x200 container is 75 pixels high, rather than 150 pixels high, like it may seem at first to anyone with common sense.
